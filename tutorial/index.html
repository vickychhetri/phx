1. Basics & Output

PHX uses a syntax derived from PHP. Programs must start with the standard <?php tag. Comments are supported using // for single lines.

Variable Declaration & Types

Variables are dynamically typed and prefixed with a $ character. Supported basic types include Integers, Floats, Booleans, Strings, Arrays, and Null.

<?php
$name = "Vicky";      // String
$age = 25;            // Integer
$pi = 3.14159;        // Float
$isCompilerDeveloper = true; // Boolean
$noValue = null;      // Null
?>

Standard Output

To print output to the console, use the echo statement or the print_r() function for arrays.

<?php
echo "Hello, PHX!\n";
echo "Name: " . $name . "\n"; // Concatenation using '.'
?>

2. Control Flow

PHX supports standard conditional branching constructs.

If / Else If / Else

<?php
$score = 85;

if ($score >= 90) {
    echo "Grade: A\n";
} else if ($score >= 80) {
    echo "Grade: B\n";
} else {
    echo "Grade: C\n";
}
?>

Switch Statement

The switch statement matches a value sequentially against several case blocks.

<?php
$day = 3;
switch ($day) {
    case 1:
        echo "Monday\n";
        break;
    case 2:
        echo "Tuesday\n";
        break;
    case 3:
        echo "Wednesday\n";
        break;
    default:
        echo "Unknown day\n";
        break;
}
?>

Ternary Operators

<?php
$status = ($score >= 50) ? "Pass" : "Fail";
echo $status . "\n";
?>

3. Loops

Loops run block statements repeatedly based on conditions.

While & Do-While

<?php
$i = 1;
while ($i <= 5) {
    echo $i . "\n";
    $i++;
}

$x = 1;
do {
    echo $x . "\n";
    $x++;
} while ($x <= 5);
?>

For Loops

<?php
for ($i = 0; $i < 5; $i++) {
    echo "Iteration " . $i . "\n";
}
?>

Array Iteration (Foreach equivalent)

Iterate indexed or associative arrays using the for syntax key-value matching.

<?php
$fruits = array("Apple", "Banana", "Cherry");
for ($i = 0; $i < count($fruits); $i++) {
    echo $fruits[$i] . "\n";
}
?>

4. Functions & Closures

Functions isolate computational logic. They can accept arguments and return values.

Custom Functions

<?php
function add($a, $b) {
    return $a + b;
}
echo add(5, 10) . "\n";
?>

Closures & Variable Scope Binding

Closures are anonymous functions. PHX supports PHP-style explicit environment capture using the use keyword.

<?php
$base = 100;
$multiplier = function($val) use ($base) {
    return $val * $base;
};
echo $multiplier(5) . "\n"; // Output: 500
?>

5. Classes & Object-Oriented Programming (OOP)

PHX supports basic class structures, inheritance, encapsulation, and class instantiation.

Class Definition & Inheritance

<?php
class Animal {
    public $name;

    public function __construct($name) {
        $this->name = $name;
    }

    public function speak() {
        return "Generic Sound";
    }
}

class Dog extends Animal {
    public function speak() {
        return "Woof";
    }
}

$dog = new Dog("Rex");
echo $dog->name . " says " . $dog->speak() . "\n";
?>

6. Exceptions

PHX translates throwing and catching exceptions directly to native panic/recovery blocks in Go.

Try-Catch Blocks

<?php
try {
    throw new Exception("Something went wrong");
} catch (Exception $e) {
    echo "Caught exception: " . $e->getMessage() . "\n";
}
?>

7. File Operations

Files are read and written using the built-in File class.

Reading & Writing Files

<?php
$file = new File();

// Write to file
$file->open("test.txt", "w");
$file->write("Line 1\nLine 2\n");
$file->close();

// Read from file
$file->open("test.txt", "r");
$lines = $file->readLines();
$file->close();

for ($i = 0; $i < count($lines); $i++) {
    echo "Read: " . $lines[$i] . "\n";
}

// Check existence & Delete
if ($file->exists("test.txt")) {
    $file->delete("test.txt");
}
?>

8. Concurrency & Channels

PHX features powerful Go-native concurrency tools built straight into the compiler language model.

Spawning Threads & Joining Results

The spawn() statement runs a closure asynchronously in a Go goroutine and returns a Thread. Call join() on the thread to block execution until the thread terminates and returns its result.

<?php
$thread = spawn(function() {
    sleep(1);
    return 100;
});

// Do other work here...

$result = $thread->join();
echo "Thread result: " . $result . "\n";
?>

Using Channels

Channels allow threads to communicate with each other safely. Create channels with channel(capacity), send values using send(chan, val), and receive values using receive(chan).

<?php
$ch = channel(5);

spawn(function() use ($ch) {
    send($ch, "Message from thread");
});

$msg = receive($ch);
echo "Received: " . $msg . "\n";
?>

9. Compiler CLI Usage

Once you write a PHX program, you can run or build it using the PHX toolchain executable.

Compile and Run:

$ bin/phx run your_program.php

Build a Standalone Binary:

$ bin/phx build -o bin/output_executable your_program.php
$ ./bin/output_executable
